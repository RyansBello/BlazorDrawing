🖼️ Blazor Simple Image Editor
A lightweight image editor built with Blazor WebAssembly, designed for simplicity and interactivity.

✨ Features:
🖊️ Draw freehand with adjustable brush size and color

🔤 Add text elements — resizable, rotatable, and removable

😄 Insert emoji (image-based) with drag, resize, rotate, and delete support

♻️ Full undo, reset, and save support

Ideal for quick annotations, sketching, and creating expressive images directly in the browser.
